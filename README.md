# Reconocimiento_facial

## 🚀 Overview

High-performance facial recognition system using deep learning for identity classification. The system processes images, extracts facial embeddings, and classifies individuals with high accuracy.

## 🧠 Approach

* Face detection and preprocessing using MediaPipe
* Feature extraction with FaceNet (embeddings)
* Classification using a neural network (MLP)

## 📊 Results

* Accuracy: 98.6%
* Dataset: 48,000+ images
* Classes: 33 individuals
